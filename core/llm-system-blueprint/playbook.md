# Playbook

## Objetivo
Transformar uso de LLM em plataforma com previsibilidade operacional.

## Regras
- Um agente por responsabilidade primária.
- Skill com entrada/saída e métrica de sucesso explícita.
- MCP como barramento padrão para ferramentas externas.
- Workflow com avaliação e escalonamento humano.
- Observabilidade obrigatória por execução.
